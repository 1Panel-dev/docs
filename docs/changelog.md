# 更新日志

## 1 版本说明


!!! Abstract ""
    **版本号说明：** 像其它优秀开源项目一样，1Panel 将保持每月发布一个新的功能版本，功能版本中如遇较为紧急或严重的 Bug，将及时推出 Bug 修复的小版本。  
    **1Panel 版本号命名规则为：** v 大版本.功能版本. Bug 修复版本，示例如下：

    - v1.0.1 是 v1.0.0 之后的 Bug 修复版本
    - v1.1.0 是 v1.0.0 之后的功能版本
