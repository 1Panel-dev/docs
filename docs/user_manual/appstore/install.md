## 安装应用

!!! Abstract ""

    在应用商店列表中选择应用进行安装

![img.png](../../img/app/appstore.png){ width="900px" }


### 安装步骤 (以 Halo 为例)

!!! Abstract ""

    1.点击安装进入应用详情

!!! Abstract ""

    2.选择版本 点击安装
    
![img.png](../../img/app/app_detail.png)

!!! Abstract ""
    
    3.填写相关参数

![img.png](../../img/app/app_param.png)


!!! Abstract ""
    
    4.点击确认，等待应用安装结束
