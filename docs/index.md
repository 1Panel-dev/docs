## 2 产品优势

!!! Abstract ""

	- **快速建站**：深度集成 Wordpress 和 Halo，域名绑定、SSL 证书配置等一键搞定；
	- **高效管理**：通过 Web 端轻松管理 Linux 服务器，包括主机监控、文件管理、数据库管理、容器管理及常用应用软件管理；
	- **安全可靠**：最小漏洞暴露面，提供防火墙和安全审计等功能；
	- **一键备份**：支持一键备份和恢复，备份数据云端存储，永不丢失。
